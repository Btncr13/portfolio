---

## 🛠️ Tecnologías utilizadas

Este portfolio ha sido construido utilizando tecnologías web nativas para garantizar la máxima velocidad de carga y evitar el peso de librerías innecesarias:

* **HTML5:** Estructura semántica para mejorar el SEO y la accesibilidad.
* **CSS3:** Flexbox, Grid, animaciones personalizadas (@keyframes) y uso de variables de color (`#ca884a` como acento).
* **JavaScript (ES6+):** Implementación de la API *Intersection Observer* para la navegación dinámica y scroll suave personalizado.
* **FontAwesome & UIcons:** Tipografía iconográfica vectorial.

---

## ✨ Características destacadas (UX/UI)

* 🎨 **Diseño "Dark Theme":** Paleta de colores optimizada (`#121212` de fondo base) para crear profundidad visual (elevación de tarjetas) y reducir la fatiga visual.
* 📱 **100% Responsive:** Adaptación fluida a dispositivos móviles, tablets y escritorio mediante Media Queries precisas.
* 🚀 **Navegación Inteligente:** Menú de navegación flotante (*sticky*) con cálculo dinámico de `scroll-padding` que resalta automáticamente la sección activa en la que se encuentra el usuario.
* ⚡ **Rendimiento:** Animaciones CSS fluidas que no bloquean el hilo principal del navegador.
